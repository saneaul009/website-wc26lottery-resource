# website-wc26lottery-resource

## 简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态资源的形式存放，不针对任何特定域名或网站，仅供技术参考与历史记录用途。

## 目录结构

```
.
├── index.html          # 入口页面（可选）
├── pages/              # 存放各个独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：主要页面归档目录，每个 HTML 文件代表一个独立页面。
- `assets/`：存放页面可能引用的公共样式、脚本或图片资源。

## 页面归档说明

仓库中的 HTML 页面均为独立归档文件，彼此之间无直接关联。每个页面可单独运行，不依赖外部服务或特定域名。

若页面引用了 `assets/` 目录下的资源，请确保相对路径正确。建议使用相对路径进行引用，以保持仓库内资源的可移植性。

## 维护说明

- 新增页面：在 `pages/` 目录下添加新的 HTML 文件，并确保其引用的资源路径正确。
- 资源更新：请勿删除或修改已被页面引用的资源文件，以免影响已有页面的正常展示。
- 版本控制：建议在提交信息中简要说明新增或修改的内容，便于追溯。

## 使用方式

可通过 GitHub Pages 或其他静态托管服务将本仓库部署为静态站点。部署后，直接访问对应 HTML 文件路径即可查看页面内容。

## 许可

本仓库内容仅供学习与归档使用。如需引用或分发，请遵守相关法律法规。
